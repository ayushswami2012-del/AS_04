# 📝 TaskFlow - Smart To-Do Dashboard

A modern and responsive **Smart To-Do Dashboard** developed as **Task 04** during my **Web Development Internship at SkillCraft Technology**.

This project helps users efficiently organize daily tasks with features like task management, search, filtering, priority levels, dark mode, progress tracking, and local storage.

---

# ✨ Features

- ✅ Add new tasks
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Search tasks instantly
- ✅ Filter tasks (All / Pending / Completed)
- ✅ Set task priority
- ✅ Add due date & time
- ✅ Progress tracking bar
- ✅ Statistics dashboard
- ✅ Dark / Light Theme
- ✅ Responsive Design
- ✅ Local Storage support
- ✅ Keyboard support (Enter key)
- ✅ Modern Glassmorphism UI
